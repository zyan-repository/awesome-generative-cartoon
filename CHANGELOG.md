## [Unreleased]

## [1.1.0] - 2025-01-13

### Added
- 17 new papers across all evaluation layers
- Enhanced coverage of comic/manga understanding benchmarks
- Expanded lip-sync and video generation evaluation papers

### Papers Added
- **Cognitive Layer:** StoryWriter, AgentArch, CreAgentive
- **Narrative Layer:** Can LLMs Generate Good Stories (Narrative Planning)
- **Visual Layer:** T2V-CompBench, T2VTextBench, VBench-2.0, VidCapBench, TheaterGen/CMIGBench, Face Consistency Benchmark, MangaVQA, CoMix, MangaDiffusion
- **Audio Layer:** EmoSSLSphere, OmniSync, SyncAnyone, SayAnything, LatentSync

## [1.0.0] - 2025-01-12

### Added
- Initial repository structure following Full-Stack Evaluation Model
- README.md with bilingual support (English/Chinese)
- Documentation for all 5 evaluation layers:
  - Cognitive Layer (Agent intelligence)
  - Narrative Layer (Story and screenplay)
  - Visual Layer (Character consistency, video)
  - Audio Layer (TTS, music, lip-sync)
  - System Layer (Evaluation toolkits)
- 40+ papers organized by evaluation layer
- Contributing guidelines with paper submission template
- Benchmarks summary quick reference table

### Papers Added
- **Cognitive Layer:** AgentBench, MultiAgentBench, RolePlayEval, GEMMAS, PersonaEval, RolePlayBench, LLM Agent Survey
- **Narrative Layer:** SCORE, DOME, StoryBench, DSR, ETVA, Story Cloze Test
- **Visual Layer:** ViStoryBench, CharaConsist, VQAScore, TIFA, DSG, CLIPScore Survey, FVD, JEDi, T2VBench, T2VWorldBench
- **Audio Layer:** EmergentTTS-Eval, SongEval, MusicEval, CLAP, Wav2Lip, SyncNet, LipFD, LSE-D, Music Metrics Survey
- **System Layer:** DeepEval, t2v_metrics, Prescene, Multimodal Eval Survey
